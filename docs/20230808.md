## ファイルパスの抽出

- システムコールがアクセスしたファイルを、straceの出力する問題を解く
- 次のワンライな0について、seqにしかけたstraceの出力から、システムコールの引数となっているファイルパスを抽出する

```bash

strace seq 10000000 | head

```

- straceの結果を、一時ファイルに出力する

```bash

strace seq 10000000 2> temp | head

cat temp | grep -oE '"(/[^/"]*)*"'


```

- `2> temp`の解説
  - `2`は標準エラー出力を表す
    - `>`はリダイレクトを表す
    - `temp`は一時ファイルの名前
    - つまり、標準エラー出力を一時ファイルに出力する
